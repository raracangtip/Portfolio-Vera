my projects

Form Login

Pada tugas ini, saya mengembangkan form login sederhana menggunakan HTML dan CSS. Form login dirancang untuk menyediakan antarmuka yang bersih dan responsif bagi pengguna saat mengakses halaman dengan akun mereka.

Portfolio

Dalam proyek ini, saya mengembangkan website portfolio menggunakan HTML, CSS, dan JavaScript untuk menampilkan karya dan keterampilan saya di bidang web development. Website ini dirancang dengan fokus pada tampilan yang menarik, navigasi yang mudah, dan fitur interaktif.

Kalkulator

Proyek ini adalah pembuatan kalkulator web sederhana namun fungsional yang dikembangkan menggunakan HTML, CSS, dan JavaScript. Kalkulator ini dibuat untuk melakukan operasi matematika dasar dengan tampilan yang intuitif dan responsif, memungkinkan pengguna untuk menghitung langsung di perangkat mereka dengan mudah.

Dashboard

Dalam proyek ini, saya mengembangkan sebuah dashboard admin berbasis web menggunakan HTML, CSS, dan JavaScript. Dashboard ini dirancang untuk memberikan visualisasi data dan kontrol yang intuitif kepada administrator dalam mengelola data dan informasi dengan mudah.

Snake Game

Game Ular ini merupakan proyek sederhana yang dibuat menggunakan HTML, CSS, dan JavaScript. Dalam game ini, pemain mengendalikan ular untuk memakan makanan dan menghindari tabrakan dengan tubuhnya sendiri. Setiap kali ular memakan makanan, ukurannya akan bertambah dan skor akan meningkat.

Diagram Penjualan

Proyek ini bertujuan untuk membuat laporan penjualan tahunan yang disajikan dengan visualisasi diagram interaktif. Proyek ini dirancang menggunakan teknologi HTML dan JavaScript untuk menciptakan tampilan yang responsif dan menarik. Data penjualan tahunan akan divisualisasikan menggunakan pustaka JavaScript populer, seperti Chart.js.
